Simulation codes

m-AIA m-AIA is a multiphysics solver developed by several researchers over the years at the Institute of Aerodynamics, RWTH Aachen University, Germany. The code is written in C++ with MPI, ans was previously known as ZFS. m-AIA includes solvers based on finite volume method supporting both cartesian cut-cells and structured cells, lattize boltzman solver, and discontinous solver. The code has been used in several research projects, and maintained using both SVN and Git.

During my postdoc phase, I worked on the extension of the finite volume solver using the structured grid to simulation jet injection in high speed crossflows. Implicit LESs were performed to develop promising and flexible flow control technique for better aerodynamics and efficiency of aerospace transport and propulsion systems.

Compact3D Compact3D is a high-order compresible CFD and CAA code based on finite difference method. The code is written in FORTRAN with MPI, and was developed during my PhD at the Institut PPRIME, University of Poitiers, France. The code was developed to perform high-order implicit LES. For better scalability on HPC systems 2D-pencil decomposition of the computational domain was adopted. The code was used to simulate temporal and spatial turbulent flows, as well as acoustic propagation over smooth wall and acoustic liner. No-slip wall, impedance boundary conditions, sponge zones, non-reflecting inlet and outlet boundary conditions were added to the code.

ArgoDG ArgoDG is a CFD and CAA code based on the discontinous galerkin method developed at the Belgian research center CENAERO . The code is written in C++ with MPI. Several researchers contribute to the code development, and the code is maintained using SVN.

During my Master thesis I extended the acoustic module of ArgoDG to compute 3D sound propagation from a turboprob aircraft engine in take-off mode. Additionally, I added the homentropic linearized Euler equations to compute acoustic propagation in swirling flows, typically for noise computation in turbomachinery. Impedance boundary conditions and non-reflecting boundary conditions were also included. Bi-linear interpolation was developed to couple the CFD and CAA solvers. Real time Fourier transformer and output plugin for probes were also integrated to the solver.

Simple CFD code for RANS simulation The code was written as part of a personal project during the first year of M.Sc. Turbulence at Ecole Centrale de Lille. The code was written in FORTRAN and uses the finite volume method to solve RANS equations. The code was used to simulate channel flows and an algebraic turbulence model with the mixing length concept was used to model the eddy-viscosity. [github repository]

Large data analysis

Ajvg_postprocessing Post-processing routines were developed to postprocess and analyse the simulation results computed on complex multi-block grids by solver m-AIA. The code was written to perform spectral and conditional analysis, spectra reconstruction and also to interpolate the results from complex multi-block grid to regular 2D grid for faster and efficient analysis.

Turbstat Turbstat is a FORTRAN code developed during my PhD to postprocess the large simulation dataset. The code is parallelized using MPI for faster calculations. The code was written to perform major statistical operations, spectral anallysis, scale decomposition analysis, and conditional analysis of both spatial and temporal flows.

BoundaryLayer_postprocessing A FORTRAN code was developed to postprocess the experimental dataset of the EU project WALLTURB. A synchronised hot-wire rake measurement was analysed to confirm the presence of large-scale structures, and their influence on the near-wall region. The code was developed to perform statistical analsysis, spectral analysis, as well as the decoupling procedure based on Hilbert transformation.

Acoustic

Ray tracing A MATLAB script was extended to work on the student project "Development of Imaging techniques for Aeroacoustics sources". 2D & 3D ray tracing model were developed and extensively tested on mixing-layer and jet flows with temperature gradient to predict the sound propagation, and the time delays were estimated. The influence of the shear layer thickness, Mach number and temperature gradient were studied.

Beam forming A MATLAB script was extended to work on the student project "Development of Imaging techniques for Aeroacoustics sources". 2D & 3D beamforming technique with Amiet correction was used to locate the aeroacoustic source in a mixing-layer and jet flows using the time delay information computed from the Ray-tracing method. Focalisation error was estimated and cross checked with the results obtained from a similar study performed using time-reversal technique.
